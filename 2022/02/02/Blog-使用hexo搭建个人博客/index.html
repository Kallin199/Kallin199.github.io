使用hexo进行博客搭建

发表于 2022-02-02 更新于 2022-02-17 分类于 Hexo

环境配置

使用hexo在本地可以方便、快速地搭建个人博客，同时依赖github可实现将个人博客推送至远端，本篇介绍在Macbook上利用hexo和node.js搭建属于自己的博客

  1. 安装node.js

    在官网下载并按步骤安装(安装完成后会一并配置npm包管理器)

  2. 安装hexo

命令行输入：npm install -g hexo-cli

环境配置验证

命令行输入node -v, hexo -v, npm -v检测是否安装成功

本地博客搭建

  1. 在本地创建一个文件夹用于存放博客的相关文件(存放位置自定义)

​ 命令行输入：makir blog

  1. 进入blog文件，使用hexo初始化博客

    命令行输入：hexo init

​ 初始化完成后会看到提示: start blogging with hexo

  1. 使用本地预览命令启动博客

    命令行输入：hexo s

  2. 新建一篇博文

​ hexo -n "blog name"，之后可以使用任意markdown编辑器书写博文

  1. 生成博文

​ hexo g

随后便可在本地博客查阅新建的博文。

部署博客

推荐使用github部署博客

  1. 在github新建仓库，注意仓库名称必须为github用户名，即<username>.github.io。
  2. 在本地blog文件下安装插件npm install --save hexo-deployer-git
  3. 打开blog/_config.yml文件，增加Deployment部分
  1. 部署到远端

​ hexo d

  1. 访问博客，使用https://username.github.io

这样一个属于自己的博客就建成啦！

可能遇到的问题

密钥配置

​ 如果之前没有在github上提交过项目，则需要配置密钥

  1. 设定git账户和邮箱，其中 username 是你的用户名，email@gmail.com是你github的登录邮箱
  1. 然后通过终端命令生成SSH Key
  1. 复制SSH Key
  1. 打开并登陆github，点击用户头像>settings>SSH and GPG keys>New SSH key 粘贴刚刚复制的密钥，就可以将本地项目推送至远端啦。
